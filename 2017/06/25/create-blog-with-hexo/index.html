常见问题

标签和分类数量显示不对

清理缓存重新生成即可（也有可能是_drafts文件夹下有文章）

修改文章的预览信息

在文章中的front matter加上description

会将description中的内容显示在预览上，这种优先级最高，不太推荐。

在文章中加上<!-- more -->

会将<!-- more -->之后的文章截断，官方推荐

修改主题配置文件参数auto_excerpt

会截断指定字符数后的文章，官方不推荐

统计服务

使用Google 分析

  1. 注册Google分析账号
  2. 编辑主题配置文件，修改字段google_analytics，值设置成你的Google跟踪ID。跟踪ID通常是以UA-开头。

使用不蒜子统计

  1. 编辑主题配置文件中的busuanzi_count的配置项。

  2. 修改后，当enable: true时，代表开启全局开关。若site_uv、site_pv、page_pv的值均为false时，不蒜子仅作记录而不会在页面上显示。
