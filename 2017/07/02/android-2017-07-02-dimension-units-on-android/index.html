Android上的尺寸单位

发表于 2017-07-02 | 分类于 Android ， Dimension Units | 阅读次数

常见尺寸单位

px

  • pixel
  • 即像素
  • 即我们常说的分辨率
  • 物理设备上真实的点

dp（dip）

  • density-independent pixel
  • One dp is always 1/160th of an inch on a device’s screen
  • 1dp等于屏幕上的1/160英寸
  • 无关于屏幕像素密度（dpi）
  • 常用做Android的widget的尺寸

sp

  • scale-independent pixel
  • 基本等同于dp（和dp的比例一样）,但当手机默认字体改变时，sp大小也会等比例缩放
  • 常用做Android的字体尺寸

pt,mm,in

  • 不是所有的设备都正常支持
  • 类似于dp,1pt等于屏幕上的1/72英寸

dpi

  • dots per inch
  • 像素密度

常见像素密度

手机

  • mdpi = 160dpi, 1dp = 1px (baseline)
  • ldpi = 120dpi, 1dp = 0.75px
  • hdpi = 240dpi, 1dp = 1.5px
  • xhdpi = 320dpi, 1dp = 2px
  • xxhdpi = 480dpi, 1dp = 3px
  • xxxhdpi = 640dpi, 1dp = 4px

实例计算

  • 1英寸(in)=2.54厘米(cm)
  • 1dp = 1/160 in = 0.15875mm
  • 红米 note 4x 分辨率 1920px X 1080px
  • 红米 note 4x 测量得长度约 10cm ≈ 4in
  • 红米 note 4x 计算得像素密度 1920px / 4in = 480 dpi
  • 红米 note 4x 为 xxhdpi 缩放比例为3，即1dp = 3px
  • 红米 note 4x 为 640dp X 360dp

电脑

  • 显示器一般常见96dpi（以前72dpi）
  • 高分屏dpi未知
  • 在Windows中选择分辨率就是在改变dpi，即虽然屏幕的分辨率没变，但是能显示的分辨率变小
  • html 页面中的px在PC端对应真实px即分辨率（Windows设置的分辨率）
